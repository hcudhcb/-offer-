题目描述
一只青蛙一次可以跳上1级台阶，也可以跳上2级……它也可以跳上n级。求该青蛙跳上一个n级的台阶总共有多少种跳法。

思路：
也是递归或者循环，推荐循环每次保存计算结果

1：1=1
2：1+1=2
3：1+2+1=4
4：1+2+4+1=8
5：1+2+4+8+1=16
6：1+2+4+8+16+1=32

可以看出规律，f(n) = 2^(n-1)

```
# -*- coding:utf-8 -*-
class Solution:
    def jumpFloorII(self, number):
        # write code here
        return 2**(number-1)
```
